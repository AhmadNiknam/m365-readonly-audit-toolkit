# Project Summary

## What this project does

This project is a read-only Microsoft 365 audit toolkit built in PowerShell for lab and portfolio use.

## What has been implemented

- Microsoft Graph connection validation
- Settings-based execution using `config/settings.json`
- User inventory retrieval
- Sign-in summary retrieval
- Group inventory retrieval
- Disabled user inventory retrieval
- Licensed user inventory retrieval
- Authentication method summary retrieval
- CSV export for all implemented datasets
- Basic project documentation
- GitHub version control and publishing

## Current output files

- `sample-output/UserInventory.csv`
- `sample-output/SignInSummary.csv`
- `sample-output/GroupInventory.csv`
- `sample-output/DisabledUserInventory.csv`
- `sample-output/LicensedUserInventory.csv`
- `sample-output/AuthenticationMethodSummary.csv`
