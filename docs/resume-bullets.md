# Resume Bullets

- Built a PowerShell-based Microsoft 365 read-only audit toolkit in a lab tenant using Microsoft Graph.
- Developed modular reporting for user inventory, sign-in activity, group inventory, disabled users, licensed users, authentication methods, directory roles, and directory role members.
- Implemented settings-based execution with JSON configuration and generated structured CSV/Markdown outputs for audit review and validation.
- Documented the solution architecture, testing notes, portfolio notes, and executive summary, then published the project to GitHub with a release for portfolio presentation.
