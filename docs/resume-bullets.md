# Resume Bullets

- Built a PowerShell-based Microsoft 365 read-only audit toolkit in a lab tenant using Microsoft Graph.
- Developed modules for user inventory, sign-in summary, group inventory, disabled user inventory, licensed user inventory, and authentication method reporting.
- Implemented settings-based execution using JSON configuration and generated CSV/Markdown outputs for audit review.
- Documented the solution architecture, testing notes, and portfolio positioning, and published the project to GitHub.
