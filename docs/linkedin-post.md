# LinkedIn Post

I recently completed a hands-on Microsoft 365 lab project using PowerShell and Microsoft Graph.

In this project, I built a read-only audit toolkit that can:

- validate Microsoft Graph connectivity
- retrieve user inventory
- retrieve sign-in summary
- retrieve group inventory
- retrieve disabled user inventory
- retrieve licensed user inventory
- retrieve authentication method summary
- retrieve directory role summary
- retrieve directory role members
- generate an executive summary
- export audit outputs to CSV and Markdown

I also structured the solution with modular scripts, JSON-based settings, project documentation, GitHub version control, and a formal v1.0.0 release.

This project helped me strengthen my practical skills in:
- Microsoft 365 administration
- PowerShell scripting
- Microsoft Graph reporting
- directory role visibility
- read-only audit design
- technical documentation
- GitHub portfolio publishing
